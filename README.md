MoodTunes
MoodTunes is a modern MERN-stack application that curates your musical experience based on your emotional state. By leveraging the YouTube Data API v3, the platform dynamically fetches and displays high-quality music videos tailored to specific moods like Focus, Energetic, Relaxed, or Melancholy.
